Redirects the signedInUser to a page if they are logged in or logged out
@param to
@param children
@param redirectIf
@example <AuthRedirectServer to={"/"} if={"logged-in"}>...</AuthRedirectServer> // Redirects to "/" if logged in, otherwise renders children
